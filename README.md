获取Hypixel API Key
-----------
使用你的正版账号登录Hypixel，在聊天框中输入"/api new"
![step1.png](https://images2.imgbox.com/0a/e6/8YHvKTjG_o.png "步骤一")
然后，你就得到了一个可用于查询玩家数据的API Key！(点击聊天框中文字可复制)
![step2.png](https://images2.imgbox.com/f8/5f/1obVAX1w_o.png "步骤二")
